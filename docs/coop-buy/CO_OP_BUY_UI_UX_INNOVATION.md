## 🎯 Core Design Philosophy

> **"신뢰는 시각화될 때 현실이 된다"**  
> "Trust becomes real when it is visualized"

**Principles:**
1. **Transparency** — All agent actions visible in real-time

## 🌈 3 Core Visual Systems

### **1. Resonance Aura — Trust Visualization**

**Concept:** Trust expressed as living light

**Visual Properties:**
```
TRUST LEVEL         COLOR           PARTICLES       INTENSITY
0-25%      Blue (Cold)       Slow            Dim
25-50%     Purple (Caution)  Medium          Moderate
50-75%     Green (Good)      Fast            Bright
75-100%    Gold (Excellent)  Very Fast       Radiant
```

**Implementation:**
- Each transaction displays an expanding aura
- Color changes as HARM scores update in real-time
- Particle intensity reflects negotiation progress
- Aura "breathes" (pulses) with agent activity

**Example:**
```
Initial (Intent submitted):      Blue Aura (25%)
Image Agent validates:           Blue → Purple (45%)
Music Agent confirms trust:      Purple → Green (65%)
HARM validation passed:          Green → Gold (95%)
Payment confirmed:               Gold Aura (Full)
```

---

### **2. Intent Wave — Meaning Propagation**

**Concept:** User intent spreads like sound waves through system

**Visual Properties:**
- User speaks/types intent
- Soundwave-like ripples emanate outward
- Each ripple represents one agent "hearing" the intent
- Ripple color = agent understanding confidence
- Ripple speed = agent response time

**Animation Sequence:**
```
Frame 1: User input → center point
Frame 2: First ripple (Intent parsed) → Luna engine (blue)
Frame 3: Perception ripple → Image Agent (green) + Music Agent (purple)
Frame 4: Negotiation ripples → Negotiator agents (yellow)
Frame 5: Validation ripple → RyuWon + AI Aurora (teal)
Frame 6: Payment ripple → Payment agents (gold)
Frame 7: Settlement ripple → Ledger update (white)
Frame 8: Analytics ripple → Performance dashboard (silver)
```

**Real-time Display:**
- Ripples overlap and interfere (like actual waves)
- Constructive interference = agent agreement
- Destructive interference = conflicting recommendations
- Final pattern = system consensus

---

### **3. Trust Constellation — Network Visualization**

**Concept:** Participants visualized as stars; trust connections as light-paths

**Visual Properties:**

**Stars (Participants):**
- Size = reputation score
- Brightness = online/active status
- Color = agent type or human category
- Twinkle pattern = recent activity

**Light-Paths (Trust Connections):**
- Thickness = transaction value
- Color gradient = trust score progression
- Animation = real-time negotiation flow
- Glow = completed transactions (memory)

**Constellation Behaviors:**
```
PHASE          VISUALIZATION
Discovery      Stars appear scattered
Connection     Light-paths draw between matching buyers/sellers
Negotiation    Paths brighten and pulse with activity
Agreement      Paths stabilize and glow steadily
Settlement     Paths fade into constellation memory
Repeat         System ready for new transaction
```

**Interactive Elements:**
- Hover over star → See participant profile + trust history
- Click path → View transaction details
- Zoom out → See entire regional ecosystem
- Time-lapse mode → Watch 24 hours in 30 seconds

## 🖼️ Component Library

### **A. Visual Elements**

**Aura Component**
```
<Aura 
  trustScore={94.75}
  stage="settled"
  animated={true}
  particleIntensity="high"
/>
```

**Intent Wave Component**
```
<IntentWave
  sourceAgent="Luna"
  targetAgents={["Image", "Music", "Negotiator-1"]}
  speed="normal"
  colorMap="confidence"
/>
```

**Trust Constellation Component**
```
<TrustConstellation
  participants={[
    {id: "user_001", type: "human", reputation: 0.92},
    {id: "seller_002", type: "agent", reputation: 0.95},
    {id: "help_003", type: "community", reputation: 0.88}
  ]}
  connections={[
    {from: "user_001", to: "seller_002", value: 50000, trust: 0.94},
    {from: "seller_002", to: "help_003", value: 5000, trust: 0.88}
  ]}
/>
```

### **B. Animation Patterns**

**Pulse** — HARM score changing
```
@keyframes harmPulse {
  0% { scale: 1; opacity: 1; }
  50% { scale: 1.05; opacity: 0.9; }
  100% { scale: 1; opacity: 1; }
}
Duration: 1.5s
```

**Flow** — Agent activity
```
@keyframes agentFlow {
  0% { offset-distance: 0%; }
  100% { offset-distance: 100%; }
}
Duration: 2s
Path: Along light-path
```

**Breathe** — Trust readiness
```
@keyframes trustBreathe {
  0%, 100% { filter: brightness(1); }
  50% { filter: brightness(1.2); }
}
Duration: 3s
```
