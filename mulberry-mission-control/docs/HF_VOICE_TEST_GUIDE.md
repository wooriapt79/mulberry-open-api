## ⚠️ 예상 가능한 문제 & 해결

### 문제 1: 업로드 실패
```
증상: "Upload failed" 에러

원인:
- 파일 크기 너무 큼 (>10MB)
- 지원하지 않는 형식
- 네트워크 문제

해결:
1. 파일 크기 확인 (<5MB 권장)
2. WAV/MP3 형식 확인
3. 다시 시도
4. 브라우저 새로고침
```

### 문제 2: 전처리 느림
```
증상: 처리 시간 >1초

원인:
- HF Spaces CPU 부하
- 파일이 너무 김 (>10초)
- 모델 로딩 중

해결:
1. 짧은 파일로 테스트 (<5초)
2. 잠시 후 재시도
3. 페이지 새로고침
```

### 문제 3: 인식 정확도 낮음
```
증상: 텍스트 이상하게 나옴

원인:
- 노이즈 너무 심함
- 음질 낮음
- 언어 설정 문제

해결:
1. 깨끗한 환경에서 재녹음
2. 마이크 가까이
3. 한국어 설정 확인
```

### 문제 4: DTMF 인식 안 됨
```
증상: DTMF 톤 인식 실패

원인:
- 톤 길이 너무 짧음
- 주파수 비표준
- 노이즈 간섭

해결:
1. 표준 DTMF 생성기 사용
2. 톤 길이 >100ms
3. 깨끗한 파일
```
