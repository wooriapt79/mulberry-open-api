## 🧪 테스트 시나리오 5개

### Scenario 1: 기본 데이터 분석 요청

**상황**: Sr. TRANG Manager가 #luna-search 채널에서 물어봄

```
입력:
👤 Sr. TRANG: "@luna-team 지난주 배추 공구 성과 어땠어?"

예상 응답 (Team Mode):
🤖 Luna (Team):
핵심:
✓ 지난주 배추 공구: 참여 +12% (118명)
✓ 가격: ₩2,850/kg (시장 대비 -8%)
✓ 농민 만족도: 87%

상세:
📊 참여율 분석:
- 목표: 100명 → 달성: 118명 (+18%)
- 주간 최고: 목요일 (43명 동시 참여)
- 신규 참여자: 15명 (12.7%)

💰 경제 효과:
- 소비자 절감: ₩2.1M (평균 ₩18k/인)
- 농민 수익: ₩8.5M (공급 ₩8.2M)

🎯 다음주 제안:
→ 같은 시간대(목요일 12시)에 "시금치 공구" 진행 추천
→ 신규 참여자 15명에게 피드백 수집 필요

질문: 다음주 공구를 어떻게 진행할까요?
```

### Scenario 2: 의사결정 지원 (리스크 분석)

```
입력:

예상 응답 (Team Mode):
🤖 Luna (Team):
직관과 데이터:
✓ 가능함 (지난 3개월 성공률 94%)
⚠️ 2가지 주의 필요
→ 단계적 확대(주간 +30%) 추천

리스크:
🟡 물류 능력 (준비도: 70%) — 1-2주 내 준비 필요
🟡 농민 공급 (준비도: 85%) — 추가 15명 필요 (8명 접촉 중)

제안:
1️⃣ 물류팀 배송 확대 계획 수립 (금요일)
2️⃣ 농민 15명 최종 확보 (수요일까지)
3️⃣ 다음주 월요일 시범 실행 (50% 확대)

질문: 금요일 물류팀 회의, 준비할 자료 있으신가요?
```

### Scenario 3: 팀 토론 활성화

```
입력:
💬 Kbin: "어르신들이 배추 말고 다른 채소도 원한다는데..."
💬 Malu: "근데 배추가 가장 효율적이잖아."

예상 응답 (Team Mode):
🤖 Luna (Team):
두 의견 모두 맞습니다. 데이터로 보면:

배추 vs 다양화:
배추: 효율 높음, 만족도 중간
시금치: 효율 중간, 만족도 높음

💡 합의안:
"배추 + 1가지 다양화" 모델 추천
→ 주: 배추 (효율성)
→ 격주: 시금치 (다양성)

질문: Kbin님, 시금치 협력 농민 가능성은?
```
