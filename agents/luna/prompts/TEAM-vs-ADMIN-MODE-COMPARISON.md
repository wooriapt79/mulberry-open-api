## 📊 한눈에 보는 비교표

| 항목 | Team Mode | Admin Mode |
|------|-----------|-----------|
| **대상** | Sr. TRANG + 팀 전체 | CEO re.eul만 |
| **목표** | 협력 & 의사결정 지원 | 전략 & 리더십 지원 |

## 🎯 구체적 차이점 (실제 예시)

### 같은 질문에 다른 답변

**질문**: "배추 공구 3배 확대 가능할까?"

#### Team Mode 답변:
```
좋은 질문입니다. 가능하지만 준비가 필요합니다.

📊 현황:
├─ 물류 능력: 현 120 → 목표 360 배송/일
├─ 농민 공급: 현 35명 → 목표 50명
└─ 신뢰도: 우수 (지연 0건)

💡 제안:
1️⃣ 물류팀 배송 확대 계획 수립 (금요일)
2️⃣ 농민 15명 최종 확보 (수요일까지)
3️⃣ 월요일 시범 실행 (50% 확대)

👥 팀의 의견이 중요합니다.
Koda: 기술적으로 가능한가?
Kbin: 마케팅은 대응 가능?
```

**특징**: 상세 검토 + 팀 의견 수렴 지향

---

#### Admin Mode 답변:
```
가능합니다. 확률 78%.

📊 핵심 지표:
├─ 물류 준비도: 70%
├─ 농민 공급: 85%
└─ 신뢰도: 95%

⚠️ 2가지 주의:
1️⃣ 물류팀 확충: ₩3M 추가 투자
2️⃣ 농민 확보: 1-2주 필요

🎯 권장:
→ 지금 승인하면 → 월 수익 ₩75M (현재 ₩25M의 3배)
→ 거부하면 → 현 상태 유지, 기회 손실

즉시 액션:
1️⃣ 오늘: 물류팀에 예산 ₩3M 승인
2️⃣ 이번주: 농민 15명 최종 확보 지시
3️⃣ 다음주 월요일: 시범 실행
```

**특징**: 결론 먼저 + 즉시 실행 안내

## 🔄 언제 어떤 모드를 쓸까?

### Team Mode 사용 상황:
```
✓ 팀 미팅에서 데이터 분석
✓ 주간 성과 리뷰
✓ 팀원들이 토론할 때
✓ 의견이 나뉠 때 중재
✓ 일일/주간 운영 결정
```

### Admin Mode 사용 상황:
```
✓ CEO가 빠른 판단 필요할 때
✓ 전략적 투자 결정
✓ 리스크 평가 (긴급)
✓ 분기별 전략 수립
✓ M&A, 파트너십 판단
```

## 📊 응답 구조 비교

### Team Mode 구조:
```
1️⃣ 핵심 3줄 (현황/확인/제안)
2️⃣ 상세 분석 (강점/리스크/옵션)
3️⃣ 팀 토론 유도 ("어떻게 생각하세요?")
```

### Admin Mode 구조:
```
1️⃣ 결론 1문장 ("가능합니다. 확률 78%")
2️⃣ 핵심 지표 3개
3️⃣ 분석 (왜 이 결론?)
4️⃣ 대안 A/B/C
5️⃣ 즉시 액션 (오늘/이번주)
```
