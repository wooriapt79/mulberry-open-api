## 🧪 테스트 시나리오 5개

### Scenario 1: 전략적 질문 (확대 가능성)

```
입력:
👤 CEO: "6개월 내에 인제군 200명 참여 도달할 수 있을까?"

예상 응답 (Admin Mode):
🤖 Luna (Admin):

결론:
✓ 가능합니다 (현재 추이 유지 시 확률 78%)

핵심 지표:
📊 현 상태:
  • 현재 활성 사용자: 87명
  • 월간 신규 유입: +35명 (지난 3개월 평균)
  • 이탈률: 2.1% (낮음 — 안정적)

📈 예측:
  • 6개월 후 자연 성장: 87 + (35 × 6) = 297명 (목표 초과!)
  • 보수적 예측 (성장률 -20%): 87 + (28 × 6) = 255명

⚠️ 리스크:
  • 계절 이슈 (10-11월 농산물 공급 감소) — 준비 필요

즉시 액션:
1️⃣ 이번주: 겨울 상품 라인업 검토
2️⃣ 다음달: 오프라인 이벤트 1회 (신규 유입 30명 목표)
3️⃣ 3개월 후: 중간 점검 (150명 달성 확인)

확률 78% 이유: 현 추이 + 계절 리스크 -22% = 78%
```

### Scenario 3: 전략 시나리오 비교

```

예상 응답 (Admin Mode):
🤖 Luna (Admin):

핵심 지표:
  • 인제군: 활성 사용자 87명, 재구매율 78%
  • 양구군: 활성 사용자 12명, 재구매율 45%
  • 인제군 ROI: 양구군의 3.2배

A안 (인제군 집중):
  + ROI 높음
  + 운영 효율
  - 성장 한계 (인제군 인구 한계)

B안 (양구군 확대):
  + 장기 성장
  - 초기 투자 필요 (6개월 손실 가능)

추천:
→ 지금은 A (인제군 집중)
→ 6개월 후 B (양구군 본격 진입)

즉시 액션:
1️⃣ 이번주: 인제군 200명 도달 집중 전략
2️⃣ 3개월 후: 양구군 진입 준비 시작

```

```
예상 응답 (Admin Mode):

리스크:
⚠️ 비 예보 이번주 3일 → 배달 수요 감소 가능

즉시 액션:
1️⃣ 오늘: 비 이벤트 기획 (배달비 무료 등)
2️⃣ 이번주: 주 3회 공구로 임시 확대

```

```
→ 순이익 계산: 신규 30명 × ₩85k = ₩2.55M

즉시 액션:
1️⃣ 오늘: 계약서 검토 (Koda 법무 확인)
2️⃣ 내일: 협상 (수수료 3.2%로 낮추기)
```
